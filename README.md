# PSA PBL Workspace

This folder contains the project workspace for the **Autonomous Explainable Grid Digital Twin (PoC)**.

## Where the actual PoC lives
- **`platform/`** — the runnable PoC modules (contracts, simulators/adapters, orchestrator, docs)
- **`upstreams/`** — vendored/cloned upstream simulator sources (OpenDSS, GridLAB-D, etc.)
