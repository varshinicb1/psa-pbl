# dt-contracts

Canonical, versioned contracts for the digital twin:
- `GridGraphSnapshot` (topology + state)
- `TelemetryTick` (measurements + events)
- `ActionPlan` (switching + setpoints)
- `ExplanationPacket` (XAI payloads + physics residuals + uncertainty)

These schemas are designed to keep the platform **modular, distributed, and future-proof**.
